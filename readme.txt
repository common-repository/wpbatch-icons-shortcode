== Description ==

This Plugin will Help you to add any kind of icons with colors, background, space, size and some more features

Please Check the Demo: <a href="http://dreamwebit.com/wpbatch-icons-shortcode" target="_blank">WPBatch Icons Shortcode</a>

**Features include:**

* **Shortcode**  You will get the opportunity to use shortcodes to show the icons.



== Installation ==

1. Upload `wpbatch-icons-shortcode.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. or you can search by <pre>wpbatch icons shortcode</pre> from dashboard>plugins>add new to install and activate
4. Place `[icon]the_name_of_the_icon[/icon]` in your pages or posts
